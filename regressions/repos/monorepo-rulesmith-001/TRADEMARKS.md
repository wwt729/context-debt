The names "rulesmith" and any associated logos, marks, and branding assets are the property of the project maintainer(s), unless explicitly stated otherwise.

## Allowed (without prior permission)

You may use the name `rulesmith` to:
- refer to the project factually
- describe compatibility (for example: "works with rulesmith")
- link to the official repository

## Not allowed (without written permission)

You may not:
- use the `rulesmith` name or logo in a way that suggests endorsement or official status
- publish modified versions under the same primary name or branding
- use confusingly similar branding for competing distributions

## Forks

Forks must use clearly differentiated names and branding unless written permission is granted by the maintainer(s).
