# Governance

`rulesmith` is a maintainer-led project.

## Project leadership

Project maintainer(s) have final authority over:
- roadmap
- architecture decisions
- release schedule
- merge and reject decisions
- project policies (including CLA requirements)

## Contributions

Contributions are welcome, but acceptance is at maintainer discretion.

## Forks

Forks are permitted under the project license. Forks must not imply official affiliation with `rulesmith`.

## Official project identity

Only maintainer-approved releases, repositories, and channels may be presented as official `rulesmith`.
